# Readme
### create presentation
1. Go to https://github.com/BChevallier/revealjs_template
2. Press "Use this template"
3. Modify slides (sections) under docs
#### localhost presentation
1. Open package.json
2. Run line format
3. Run line dev
### Host with GitHub Pages
1. Go to settings (in GitHub)->Pages and ensure "sources" is set to "GitHub Actions"
2. Check if the build was successful
3. Ensure that "base" in "vite.config.ts" is set to the repository name
